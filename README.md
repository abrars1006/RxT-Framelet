# RxT Framelet 📸

Your dark cinematic portfolio website is ready! 

## Features Integrated
- **Mobile First Design**: Looks stunning on iPhones and Androids, scales beautifully to desktop.
- **Cinematic Theme**: Dark UI (`#0F0F0F`, `#1A1A1A`) with striking red UI components.
- **Smooth Animations**: Intersection observers are used to create clean professional fade-in effects as the user scrolls.
- **WhatsApp Integration**: The `booking.html` page perfectly formats form fields into a WhatsApp text enquiry.

## How to use
1. **View the site:** Simply click and open `index.html` in your favorite browser.
2. **Setup Founder Image:** To display your photo in the "Meet The Creator" section, place your image into the folder `images/` within this project, and name it exactly `founder.jpg`.
